Проект сделан в Visual Studio 2022. Административная часть (в данный момент доделана)
представляет собой список напитков с иконками удаления и изменения. В данный момент использована база данных
sql express 2022. Сам проект построен по шаблону .net core mvc 7, один контроллер, два сервиса. Использован подход code-first.
В качестве css фреймворка выбран bootstrap 5, также использованы bootstrap-icons

Работа административной части построена на всплывающих модальных окнах.
Используется bootstrap 5. Также применена валидация из того же bootstrap.
Доступ в панель администратора доступен с параметром secretKey=SuperSecretKey, иначе - NotFound().

В клиентской части все требования соблюдены.

Роутинг:

localhost:port - панель администратора
localhost:port/drinks - магазин

Учтена обработка ошибок при внесении некорректных значений при обновлении/добавлении/удалении напитков. При покупке напитка меняется
его количество и на клиенте, и в базе, реализация через fetch.

Единственное - не удалось подключить БД в папку App_Data, развернул локально в подключенных службах.
