Проект сделан в Visual Studio 2022. Административная часть (в данный момент доделана)
представляет собой список напитков с иконками удаления и изменения. В данный момент использована база данных
sql express 2022. Сам проект построен по шаблону .net core mvc 7, один контроллер, два сервиса. Использован подход code-first.

Работа административной части построена на всплывающих модальных окнах.
Используется bootstrap 5. Также применена валидация из того же bootstrap.
В данный момент все работает корректно, единственное, база подключена не из папки App_Data, а развернута на локальном компьютере.
Доступ в панель администратора доступенс параметром secretKey=SuperSecretKey, иначе - NotFound().

В данный момент административная часть полностью готова.
